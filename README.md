- 15:31~16:59: 한국 종가 고정
- 17:00~22:29: 미국 프리마켓 기반 예측

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

- ETF 이름 클릭 → 바로 아래 구성종목 상세 펼침 구조만 남겼습니다.


## v0.5.7 업데이트

사용자가 제공한 최신 구성종목/보유주식수를 반영했습니다.

- KODEX 미국AI전력핵심인프라: VST, SMR 제거 / POWL, FIX 추가
- TIGER 미국우주테크: SPCE 제거 / VOYG 추가
- 1Q 미국 우주항공테크: VRT 제거 / BETA, LUNR, PLTR, PL, LMT, NOC 추가

비중이 제공되지 않은 종목은 `weight=0`으로 두고, 앱 실행 시 `보유주식수 × 현재가` 기준으로 실시간 계산비중을 재계산합니다.


## v0.5.7 변경사항

- KODEX 미국AI전력핵심인프라, TIGER 미국우주테크 2개 ETF만 남김
- 다른 우주항공 ETF는 앱 화면과 계산 대상에서 제거
- 최신 제공 구성종목/보유주식수 유지
