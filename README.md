# 둥이 ETF 예측기 v0.5.5 Clean List

## 핵심 변경점

한국시간 기준으로 모드를 자동 전환합니다.

- 09:00~15:30: 한국장 실제 ETF 가격 반영
- 15:31~16:59: 한국 종가 고정
- 17:00~22:29: 미국 프리마켓 기반 예측
- 22:30~04:59: 미국 정규장 기반 예측
- 05:00~08:59: 미국 애프터마켓 기반 예측

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```
